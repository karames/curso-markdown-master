# Encabezados

## Uso

- Para su uso en títulos.
- Formatea en negrita y con difentes tamaños.
- Inserta automáticamente un salto de línea.
- Equivalente a las etiquetas de encabezado en HTML (`<h1>, <h2>, <h3>...<h6>`)

## Sintaxis

- Utilizar símbolo #, espacio y título.
- `# Título`

## Ejemplos

```md
# Encabezado h1

## Encabezado h2

### Encabezado h3

#### Encabezado h4

##### Encabezado h5

###### Encabezado h6
```
