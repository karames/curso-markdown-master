# Listas desordenadas

## Uso

- Permite mostrar una lista de items con viñetas.

## Sintaxis

- Aconsejable utilizar "-" aunque se puede comenzar con "\*", "+" o "-", y a continuación [ESPACIO]
- Niveles de sangrado con [TAB].

```md
- Elemento 1
- Elemento 2
  - Elemento 21
  - Elemento 22
- Elemento 3
```

## Ejemplos

- Elemento 1
- Elemento 2
  - Elemento 21
  - Elemento 22
- Elemento 3
